Agent instruction — copy & paste
내 TON AI Access 설정을 도와줘. 아래 명령을 실행해서 Claude Code와 Codex에서 이 API 키를 쓸 수 있도록 환경 변수를 설정해줘.
TON_API_KEY="sk-litellm-xK9mQpR7vT2nBfLw4JhYcD8eZ1aUoI3s" \ TON_MODEL="qwen-3.6" \ bash <(curl -fsSL https://ton-ai-access.vercel.app/configure-cli.sh) \ --non-interactive
설정이 끝나면 source ~/.zshrc (또는 ~/.bashrc)를 실행해서 현재 세션에도 적용해줘.

스크립트는 idempotent합니다 — 두 번 실행해도 블록이 중복되지 않습니다. ANTHROPIC_API_KEY, ANTHROPIC_BASE_URL, OPENAI_API_KEY, OPENAI_BASE_URL이 자동으로 설정됩니다. 설치 가이드 →
