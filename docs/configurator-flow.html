TON AI Access — Configurator Flow

configure-cli.sh 실행 시 각 CLI별 설정 변경 흐름

Entry Point
configure-cli.sh 실행
로컬 또는 원격 실행
bash scripts/configure-cli.sh bash <(curl -fsSL …/configure-cli.sh)
① Target 자동 감지 — detect_target()
1 --target <claude|codex|openclaw|hermes|all> 명시적 지정 → 그대로 사용
2 $CLAUDE_CODE_ENTRYPOINT 설정됨 → claude
3 $CODEX_SHELL 설정됨 → codex
4 $OPENCLAW_HOME 설정됨 → openclaw
5 $HERMES_HOME 설정됨 → hermes
6 ppid 프로세스명 (codex / openclaw / hermes) → 해당 타겟
7 감지 실패 → all (전부 설정)
② 모델 선택
Non-Interactive
TON_MODEL 지정됨
TON_MODEL 환경변수를 그대로 사용
별도 프롬프트 없음
Non-Interactive
TON_MODEL 미지정
GET /v1/models 호출
번호 메뉴 표시 → read 입력
탐색 실패 시 텍스트 입력 폴백
기본값: qwen-3.6
Interactive
API key 입력 → GET /v1/models
번호 메뉴 표시 → read 입력
base URL 확인
기본값: qwen-3.6
③ Shell Profile 업데이트 — 공통 (멱등성 보장)
~/.zshrc 또는 ~/.bashrc 에 블록 삽입 (기존 블록 자동 교체)
Claude / all 타겟
ANTHROPIC_API_KEY=$API_KEY
ANTHROPIC_BASE_URL=$BASE_URL
ANTHROPIC_MODEL=$MODEL
ANTHROPIC_SMALL_FAST_MODEL
ANTHROPIC_DEFAULT_HAIKU_MODEL
ANTHROPIC_DEFAULT_SONNET_MODEL
ANTHROPIC_DEFAULT_OPUS_MODEL
↑ 모두 $MODEL로 통일
Codex / all 타겟
OPENAI_API_KEY=$API_KEY
OPENAI_BASE_URL=$BASE_URL/v1
OpenClaw / Hermes 타겟:
쉘 프로파일 env 블록 없음
(config 파일로만 설정)
🤖 Claude Code --target claude
④ config 파일 수정
~/.claude/settings.json
env.ANTHROPIC_API_KEY
env.ANTHROPIC_BASE_URL
env.ANTHROPIC_MODEL
env.ANTHROPIC_*_MODEL × 4
⊕ JSON 병합 (Python)
기존 키 보존, env 블록만 갱신
⑤ 적용 방식
현재 세션
→ export 즉시 적용
새 터미널
→ settings.json env 블록
(shell profile 불필요)
⑥ 사용처
claude CLI 기동 시
settings.json env →
ANTHROPIC_* 자동 주입
LiteLLM 프록시 경유
⌨️ Codex CLI --target codex
④ config 파일 수정
~/.codex/config.toml
model="$MODEL"
model_provider="tokamak"
[model_providers.tokamak]
base_url="$BASE_URL/v1"
env_key="OPENAI_API_KEY"
↺ 파일 전체 덮어쓰기
⑤ 적용 방식
API key 조회 경로
env_key = OPENAI_API_KEY
→ shell profile에서 주입
새 터미널 후 적용
⑥ 사용처
codex CLI 기동 시
config.toml → base_url
OPENAI_API_KEY → key
LiteLLM 프록시 경유
🦅 OpenClaw --target openclaw
④ config 파일 수정
~/.openclaw/openclaw.json
models.providers.litellm
baseUrl:"$BASE_URL"
apiKey:"$API_KEY"
api:"openai-completions"
models.default:"litellm/$MODEL"
⊕ JSON 병합 (Python)
기존 설정 보존, litellm 블록만 갱신
⑤ 적용 방식
API key + base_url
openclaw.json에 직접 저장
(shell profile env 없음)
재시작 없이 즉시 적용
⑥ 사용처
openclaw CLI 기동 시
openclaw.json → provider
models.default 우선
LiteLLM 프록시 경유
🪽 Hermes --target hermes

⑤ 적용 방식

(shell profile env 없음)
재시작 없이 즉시 적용
⑥ 사용처
hermes CLI 기동 시
config.yaml → base_url

LiteLLM 프록시 경유
