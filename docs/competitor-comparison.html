일일 cap = abuse 차단 자리 — Basic 2M/일은 Cerebras 무료(1M/일)의 2배. "유료지만 무료보다 후함" 성립.

TPM (토큰/분) · 로그 스케일

버스트 처리량. 에이전트 워크로드가 가장 먼저 막히는 차원.

일일 토큰 cap (토큰/일) · 선형

무료 티어의 핵심 throttle. 우리가 추가할 주력 차등·abuse 차단 레버.

RPM (요청/분) · 선형

버스트 빈도. 단, 에이전트는 보통 RPM보다 TPM이 먼저 막힘 → 차등 보조 레버.

1인 자가 호스팅 월 비용 vs 우리

"직접 GPU 돌리기"의 월 비용($, 로그 스케일). 클라우드 GPU 24/7(730h/월) 임대 기준.

해석: 1인이 로컬 모델을 직접 돌리면 월 $70–1,800+(임대) 또는 $1,600+ 선투자(소유). 우리는 한도 안에서 $5–10 — "충분하면" 10~300배 저렴. 단 자가 호스팅의 전용·무제한·프라이버시·임의 모델은 우리가 대체 못 함(정직한 트레이드오프). 우리의 자리 = "직접 돌릴 필요/비용 없이 충분한 모델을 싸게."
경쟁사 값은 무료/저가 티어 기준이며 수시로 변동. TPM은 사별 per-key/per-org 기준이 달라 완전 동일 비교 아님. 미공개 항목은 "비공개" 표기. TON Basic/Plus 한도는 제안치 — 베타 사용량으로 보정 필요. 일일 cap(TPD)은 LiteLLM 네이티브 한계 시 게이트웨이 측 enforce 필요. GPU 임대가는 클라우드 GPU 시장가(2026, 730h/월 환산)이며 spot/on-demand·공급자별 편차 큼. QwenCloud 가격은 페이지 직접 확인($30/$100/$200, 2026-06).
